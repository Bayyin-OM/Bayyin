# بيّن — هيكل المشروع المُنظَّم

## البنية الكاملة

```
bayyin/
├── index.html          ← الصفحة الرئيسية (Landing + Auth + Grade Picker)
├── style.css           ← كل CSS (1,837 سطر)
├── darkmode.js         ← سكريبت الوضع الداكن (يُحقَن في <head>)
│
├── grades/
│   ├── grade5.html     ← الصف الخامس  (الضوء، الظلال، الأرض)
│   ├── grade6.html     ← الصف السادس  (القوى، الكهرباء)
│   ├── grade7.html     ← الصف السابع  (البيئة، الكائنات، الأحماض)
│   ├── grade8.html     ← الصف الثامن  (المادة، المغناطيسية، الجسم)
│   └── grade9.html     ← الصف التاسع  (كيمياء، فيزياء، أحياء)
│
└── sims/
    ├── core.js             ← المحرك الأساسي (التنقل، SIM_META، محرك المحاكاة)
    ├── auth.js             ← Supabase Auth + نظام الاشتراك
    ├── unit6.js            ← محاكاة الكهرباء والموصلات (الصف السادس)
    ├── unit7.js            ← محاكاة البيئة والكائنات (الصف السابع)
    ├── unit8.js            ← محاكاة خصائص المادة (الصف السابع/الثامن)
    ├── unit9.js            ← محاكاة القوى والحركة (الصف السادس/السابع)
    ├── unit10.js           ← محاكاة التباين والتصنيف (الصف السابع)
    ├── unit11_acids.js     ← محاكاة الأحماض والقواعد (الصف السابع)
    ├── unit11_magnets.js   ← محاكاة المغناطيسية والكهرباء (الصف الثامن)
    ├── grade5_sims.js      ← كل محاكاة الصف الخامس (الضوء/الظلال/الأرض)
    ├── grade8_extra.js     ← محاكاة إضافية الصف الثامن (القلب، الأملاح، الصوت)
    └── grade9_chem.js      ← كل محاكاة الصف التاسع (كيمياء + بيئة)
```

## التدفق (Flow)

```
index.html
  ├── Landing Page  → [ابدأ الآن] → Grade Picker
  └── Grade Picker  → grade5.html / grade6.html / ... / grade9.html
                              ↓
                       Sim Panel (محاكاة تفاعلية)
```

## ملاحظات مهمة

### لماذا تحمّل كل ملفات JS في كل صفحة؟
المحاكاة تعتمد على `startSim()` الموجودة في `core.js` التي تستدعي دوالاً من جميع ملفات
الوحدات. لتجنب أخطاء "function not defined"، جميع ملفات JS مُحمَّلة في كل صفحة.
في الإنتاج يمكن تجميعها بـ Webpack/Rollup.

### الوضع الداكن
`darkmode.js` يُحقَن في `<head>` قبل رسم الصفحة لمنع وميض الألوان.

### Auth (Supabase)
مفاتيح Supabase موجودة في `sims/auth.js`. لا تنشرها في مستودع عام.

### المتطلبات
- اتصال بالإنترنت (Supabase + Google Fonts + Tajawal)
- متصفح حديث يدعم Canvas API و Web Audio API
